AI Resume Screening & ATS Score Predictor

An AI-powered Resume Screening and ATS Score Prediction system built using Machine Learning. This project helps automate the resume screening process by predicting whether a candidate is shortlisted based on resume-related features.

📌 Project Overview

This project uses Machine Learning algorithms to analyze candidate data and predict resume selection. Different models are trained and compared to identify the best-performing algorithm.

Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Training
- Model Accuracy Comparison
- Feature Importance Analysis
- Resume Prediction
- ATS Score Prediction

Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab
- GitHub

Machine Learning Models

- Logistic Regression
- Decision Tree
- Random Forest

Model Performance

| Model | Accuracy |
|--------|----------|
| Logistic Regression | 90% |
| Decision Tree | 86% |
| Random Forest | 90% |

Best Model:Random Forest

Dataset

Dataset Name:AI_Driven_Resume_Screening.csv

The dataset contains candidate information such as:

- Years of Experience
- GitHub Activity
- Project Count
- Skills Match Score
- Resume Length
- Education Level
